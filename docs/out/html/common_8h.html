Retrieves the runtime type name of a type.

Tries to retrieve the runtime type name for template parameter T.
On GCC and Clang, this will also demangle the symbol.

Template Parameters

Parameters

Returns

◆ stynamic_cast() [1/2]

template<class Derived , class Base >
constexpr auto extrait::stynamic_cast ( Base & base) -> std::remove_reference_t<Derived>&
inlineconstexprnoexcept

Downcasts an object without the overhead of dynamic_cast with a debug runtime type safe-guard.

Downcasts class object base of type Base to class Derived with static_cast and asserts in debug builds if Base is not an instance of Derived.

Template Parameters
DerivedA pointer or reference to a type that is a subclass of Base (taking cv-qualification into account)
BaseThe type of the base class to cast to Derived, this is usually automatically deduced
Parameters
baseA pointer or reference to an instance of a virtual base class
Returns
The converted type that would result from casting the Base objet to Derived
