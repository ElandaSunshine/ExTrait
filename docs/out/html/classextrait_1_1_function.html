extrait::Function< Func > Class Template Reference

Detailed Description

template<auto Func>
class extrait::Function< Func >

Describes a function pointer and its attributes.
With this class you can request any (currently attainable) information about function pointers, such as their different qualifications, their ownership, return type, parameter types and so on.

This will not just work with functions that have overloads, for these cases there is the extrait::Overload class, which allows defining the function signature in question:

using extrait::Function, extrait::Overload; // Simplify our access for illustration purposes
void overloadedFunc();
void overloadedFunc(int);
using Func1 = Function&lt;Overload&lt;void()&gt;::of(overloadedFunc)&gt;; // Works, we get the first overload "void overloadedFunc();"
using Func2 = Function&lt;Overload&lt;void(int)&gt;::of(overloadedFunc)&gt;; // Works, we get the second overload "void overloadedFunc(int);"
using Func = Function&lt;overloadedFunc&gt;; // Does not work, we don't know the specific overload
