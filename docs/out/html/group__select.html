Detailed Description

Selects a type among a set of conditions.

Evaluates a list of extrait::branch in Branches and provides a member alias "type" with the branch's associated type if the branch's condition parameter specified true, otherwise will continue to evaluate any remaining branches until one that is true.
If none of the given branches evaluates to true, a fallback type can be specified as the last type in list Branches, which can be any type that is not extrait::branch, if no such fallback was provided compilation will fail.

If at any position in list Branches (except last), is not an instantiation of type extrait::branch, compilation will fail.
Compilation will also fail if no argument was given to Branches, if list does not provide any instantiation of type extrait::branch but one element of any other type, this type will always be selected.

Adding custom template specialisations to this template results in undefined behaviour.

Template Parameters
BranchesA list of extrait::branch instantiations, optionally followed by one non branch type as fallback
