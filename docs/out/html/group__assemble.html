Detailed Description

Assembles a parameter-list based on conditions.

Evaluates instantiations of extrait::branch in list Branches and provides a new instantiation of class template T with a parameter-list consisting of the associated branch types that evaluated to true and all given non-branch types.
If a type in list Branches is a non-branch type, this type will inevitably end up in the resulting parameter-list at the position it was specified in.

Adding custom template specialisations to this template results in undefined behaviour.
